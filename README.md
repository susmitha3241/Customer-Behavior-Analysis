# Customer Behavior Analysis Dashboard

## Project Overview
This project analyzes customer shopping behavior using Python, SQL, and Power BI.  
The goal is to understand customer purchasing patterns, revenue trends, product performance, and customer segments.

## Tools Used
- Python (Pandas, Data Analysis)
- MySQL (SQL Queries)
- Power BI (Dashboard Visualization)
- Excel/CSV Dataset

## Dataset
Dataset: Customer Shopping Behavior Dataset

The dataset contains customer details such as:
- Customer ID
- Age
- Gender
- Product Purchased
- Category
- Purchase Amount
- Review Rating
- Subscription Status
- Shipping Type

## Analysis Performed Using SQL

1. Revenue analysis by gender
2. Product rating analysis
3. Top rated products identification
4. Shipping type purchase comparison
5. Subscription customer spending analysis
6. Category wise top products analysis
7. Age group revenue analysis

## Python Analysis
Performed:
- Data loading
- Data cleaning
- Data exploration
- Basic analysis

## Power BI Dashboard
Created an interactive dashboard to visualize:
- Total customers
- Revenue analysis
- Customer behavior trends
- Product insights
- Category performance

## Project Files

- Dataset folder → CSV file
- Python folder → Jupyter Notebook
- SQL folder → SQL queries
- PowerBI folder → Dashboard file

## Conclusion
This project provides insights into customer purchasing behavior and helps understand business trends using data analytics techniques.
